# Web Application Tutorials

Welcome to the **Web Application Tutorials** section. These step-by-step guides will help you learn how to build complete web applications using ACENji's Website Builder.

---

## Overview

These tutorials cover the complete application development lifecycle:

1. **Setting Up Pages** - Create and organize your application pages
2. **Adding Workflows** - Build interactive functionality
3. **Roles and Permissions** - Control access and security
4. **Publishing** - Deploy your application

---

## Available Tutorials

### Setting Up Pages

Learn how to create and organize your application's page structure, add tiles, and design effective layouts.

**What you'll learn:**
- Creating new pages
- Adding tiles from templates
- Arranging elements
- Setting up navigation

[Start the Setting Up Pages tutorial](./setting-up-pages/index)

---

### Adding Workflows

Discover how to add interactivity to your application through events, actions, and workflow logic.

**What you'll learn:**
- Configuring element events
- Creating action sequences
- Using variables
- Building dynamic behavior

[Start the Adding Workflows tutorial](./adding-workflows/index)

---

### Roles and Permissions

Learn to implement access control to secure your application and show different content to different users.

**What you'll learn:**
- Understanding roles
- Setting page access
- Element-level permissions
- Role-based content

[Start the Roles and Permissions tutorial](./roles-and-permissions/index)

---

### Publishing

Learn how to publish your application and share it with users.

**What you'll learn:**
- Preview your application
- Publish to production
- Configure domains
- Share with users

[Start the Publishing tutorial](./publishing/index)

---

## Prerequisites

Before starting these tutorials, you should:

1. Have an ACENji account
2. Be familiar with the Dashboard
3. Have basic understanding of web concepts

---

## Getting Help

If you get stuck during any tutorial:

1. Review the relevant documentation section
2. Check the troubleshooting tips in each tutorial
3. Contact support if needed
