# Email Element

The **Email** element displays an email action indicator or contact information on your page.

---

## Overview

Email is used for:
- Email action placeholders
- Contact information display
- Email-related indicators
- Communication actions

---

## Adding an Email Element

1. Open the **Website Builder**
2. Find **Email** in the Elements panel
3. Drag it onto your page
4. Configure properties in the **Floating Design Panel**

---

## Properties

### General Tab

| Property | Description |
|----------|-------------|
| **Question Text** | Label or description text |

### Color Tab

| Property | Description |
|----------|-------------|
| **Background** | Container background |
| **Icon Color** | Email icon color |
| **Text Color** | Label text color |

### Dimensions Tab

| Property | Description |
|----------|-------------|
| **Font Size** | Text size |
| **Padding** | Inner spacing |
| **Margin** | Outer spacing |

## Use Cases

### Contact Indicator
```
Label: "Contact Us"
Purpose: Indicate email contact option
```

### Email Action
```
Label: "Send Email"
Purpose: Email action placeholder
```

### Communication Section
```
Label: "Email Support"
Purpose: Support contact indicator
```

---

## Best Practices

1. **Combine with events** - Use with Open URL event for mailto links

2. **Label clearly** - Make the purpose obvious

3. **Position appropriately** - Place where users expect contact options

4. **Style consistently** - Match your app's icon style
