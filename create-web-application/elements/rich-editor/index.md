# Rich Editor Element

The **Rich Editor** element provides a text editor with formatting toolbar for creating rich text content.

---

## Overview

Rich Editor is ideal for:
- Formatted content creation
- Blog-style entries
- Detailed descriptions
- Rich text messages
- Any content needing formatting

---

## Adding a Rich Editor Element

1. Open the **Website Builder**
2. Find **Rich Editor** in the Elements panel
3. Drag it onto your page
4. Configure properties in the **Floating Design Panel**

---

## Properties

### General Tab

| Property | Description |
|----------|-------------|
| **Placeholder Text** | Hint text in empty editor |

### Dimensions Tab

| Property | Description |
|----------|-------------|
| **Height** | Editor height |
| **Width** | Editor width |
| **Padding** | Inner spacing |

## Page Limit

> **Note:** Only **one Rich Editor** is allowed per page. This constraint ensures performance and prevents conflicts.

---

## Rich Editor vs Text Area

| Feature | Rich Editor | Text Area |
|---------|-------------|-----------|
| Formatting | Full toolbar | None |
| Output | HTML | Plain text |
| Limit | 1 per page | Unlimited |
| Best for | Formatted content | Simple text |

---

## Best Practices

1. **Use for appropriate content** - Rich formatting isn't always needed

2. **Set useful placeholders** - Guide users on expected content

3. **Consider output** - Rich text stores as HTML

4. **Test rendering** - Verify formatted content displays correctly

5. **One per page** - Plan layouts with the single-editor limit
