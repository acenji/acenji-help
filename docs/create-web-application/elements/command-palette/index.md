# Command Palette

The **Command Palette** element provides a spotlight-style search interface with categorized commands, keyboard shortcuts, and live filtering. Similar to the Cmd+K experience in modern apps.

---

## Features

- **Search input** with keyboard shortcut hint (⌘K)
- **Categorized results** grouped by type
- **Live filtering** as user types
- **Keyboard shortcuts** displayed per command
- **Light and dark** theme support
- **Icons and descriptions** for each command

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Highlight color for active result | `#4285F4` |
| **Theme** | Color theme: light or dark | `light` |
| **Placeholder** | Search input placeholder text | `"Type a command or search..."` |
| **Show Shortcuts** | Display keyboard shortcuts on commands | `true` |
| **Max Results** | Maximum number of results to display | `8` |

---

## Commands

Commands are organized in groups. Each group has:
- **Category** — group name (e.g., "Pages", "Actions")
- **Items** — list of commands, each with title, icon, and optional shortcut

---

## Use Cases

- Quick navigation within apps
- Action launchers
- Search interfaces
- Keyboard-first workflows
