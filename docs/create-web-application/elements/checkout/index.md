# Checkout

The **Checkout** element provides a complete checkout form with shipping information, payment details, order summary, and step indicators.

---

## Features

- **Step indicator** — Shipping, Payment, Review progress bar
- **Shipping form** — name, email, address fields
- **Payment form** — card number, expiry, CVV fields
- **Order summary** sidebar with item list and totals
- **Place Order** button with customizable text
- **Two-column or stacked** layout options

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Active step and button color | `#4285F4` |
| **Show Order Summary** | Display order summary panel | `true` |
| **Show Step Indicator** | Display step progress bar | `true` |
| **Button Text** | Text for the submit button | `"Place Order"` |
| **Layout** | Form layout: two-column or stacked | `two-column` |

---

## Use Cases

- E-commerce checkout pages
- Subscription sign-up flows
- Payment forms
- Multi-step purchase wizards
