# Feature Grid

The **Feature Grid** element displays a responsive grid of feature cards, each with an icon, title, and description. Ideal for marketing and landing pages.

---

## Features

- **Responsive grid** with configurable columns (1-4)
- **MDI icons** for each feature
- **Three card styles** — flat, elevated (shadow), bordered
- **Accent color** for icon highlighting

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Columns** | Number of grid columns (1-4) | `3` |
| **Accent Color** | Color for feature icons | `#4285F4` |
| **Show Icons** | Display icons on feature cards | `true` |
| **Card Style** | Visual style: flat, elevated, or bordered | `elevated` |

---

## Features List

Each feature has:
- **Icon** — MDI icon class (e.g., "mdi-lightning-bolt")
- **Title** — feature name
- **Description** — brief description text

Click **+ Add** to add features. Click **x** to remove.

---

## Use Cases

- Product feature showcases
- Service overview pages
- Landing page sections
- "Why choose us" blocks
