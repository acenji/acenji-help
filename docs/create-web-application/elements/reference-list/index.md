# Reference List

The **Reference List** element displays a formatted list of citations, sources, or bibliography entries with author, title, year, pages, notes, and optional links.

---

## Overview

Build academic-style reference lists or source citations directly in your page. Each entry supports structured fields and clickable URLs. Number formatting is configurable.

---

## Adding a Reference List

1. Open the **Website Builder**
2. Find **Reference List** in the Elements panel (under Display)
3. Drag it onto your page
4. Add references in the **General Tab**

---

## Properties

### General Tab

| Property | Description | Default |
|----------|-------------|---------|
| **Section Title** | Heading displayed above the list | Empty |
| **Number Style** | Numbering format: bracket, dot, or parenthesis | `bracket` |
| **Show Links** | Display clickable link for entries with a URL | `true` |

### Reference Entry Fields

Each reference entry supports:

| Field | Description |
|-------|-------------|
| **Author** | Author name(s) |
| **Title** | Publication or source title (rendered in italics) |
| **Pages** | Page range (displayed as "pp. ...") |
| **Year** | Publication year (displayed in parentheses) |
| **Note** | Additional note (displayed after an em dash) |
| **URL** | Link to the source (shown as clickable "[link]") |

### Number Styles

| Style | Example |
|-------|---------|
| **Bracket** | [1], [2], [3] |
| **Dot** | 1., 2., 3. |
| **Parenthesis** | 1), 2), 3) |

### Data Source

| Property | Description | Default |
|----------|-------------|---------|
| **Data Source** | Where references come from: Static or From Element | `static` |
| **Source Element ID** | Element ID to pull references from (when From Element) | Empty |
| **Content Field** | Field name on the source element | `references` |

### Managing References

- Click **+ Add** to add a new reference entry
- Fill in the author, title, pages, year, note, and URL fields
- Click **x** to remove a reference
- References are displayed in the order they appear in the list

---

## Use Cases

- **Academic content** -- bibliography and citation lists
- **Research pages** -- link to source papers or articles
- **Legal documents** -- footnotes and source references
- **Documentation** -- further reading sections with structured entries
