# Cart

The **Cart** element displays a shopping cart with item list, quantity controls, pricing breakdown, and a checkout button.

---

## Features

- **Cart header** with item count badge
- **Item list** with image, name, quantity controls, price, and remove button
- **Coupon code** input with apply button
- **Pricing breakdown** — subtotal, shipping, tax, total
- **Checkout button** with customizable text

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Button and badge color | `#4285F4` |
| **Show Coupon Input** | Display coupon code field | `true` |
| **Show Shipping** | Show shipping cost line | `true` |
| **Show Tax** | Show tax line | `true` |
| **Checkout Button Text** | Text for the checkout button | `"Proceed to Checkout"` |

---

## Use Cases

- Shopping cart pages
- Order review screens
- E-commerce checkout flows
- Cart sidebar widgets
