# Team Card

The **Team Card** element displays a grid of team member cards with avatars, names, roles, bios, and social media links.

---

## Features

- **Member grid** with configurable columns
- **Avatar** — image or colored initials circle
- **Social icons** row (LinkedIn, Twitter, GitHub, Dribbble)
- **Two styles** — card (shadow) or minimal (flat)
- **Optional bio** text per member

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Columns** | Number of grid columns (1-4) | `3` |
| **Accent Color** | Color for avatar background | `#4285F4` |
| **Show Bio** | Display bio text on cards | `true` |
| **Show Social** | Display social media icons | `true` |
| **Card Style** | Visual style: card or minimal | `card` |

---

## Members

Each member has:
- **Name** — person's name
- **Role** — job title
- **Bio** — short bio text
- **Avatar URL** — profile image (shows initials if empty)

Click **+ Add** to add members. Click **x** to remove.

---

## Use Cases

- About us / team pages
- Company directory
- Speaker profiles for events
- Advisory board displays
