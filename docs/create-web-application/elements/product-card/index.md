# Product Card

The **Product Card** element displays an e-commerce product with image, title, description, pricing, ratings, and an add-to-cart button.

---

## Features

- **Product image** placeholder with category badge
- **Sale badge** with crossed-out original price
- **Star rating** display with review count
- **Add to Cart** button with customizable text
- **Category badge** in top-right corner

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Button and badge highlight color | `#4285F4` |
| **Show Rating** | Display star rating row | `true` |
| **Show Sale Price** | Show sale badge and original price | `true` |
| **Show Category** | Display category badge on image | `true` |
| **Button Text** | Text for the add-to-cart button | `"Add to Cart"` |

---

## Use Cases

- E-commerce product listings
- Product catalog pages
- Featured product displays
- Shopping app interfaces
