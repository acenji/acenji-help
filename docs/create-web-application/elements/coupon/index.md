# Coupon

The **Coupon** element displays a discount coupon card or a promo code input field. Choose between a visual coupon card with copy-code functionality or a simple input-and-apply style.

---

## Features

- **Card style** — visual coupon with dashed border, discount amount, code box, expiry, and copy button
- **Input style** — text field with "Apply" button for entering promo codes
- **Discount text** display (e.g., "20% OFF")
- **Expiry date** display
- **Description** text for coupon conditions

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Border and button color | `#4285F4` |
| **Style** | Display style: card or input | `card` |
| **Discount Text** | Main discount label | `"20% OFF"` |
| **Show Expiry** | Display expiration date | `true` |
| **Show Description** | Display condition text | `true` |

---

## Use Cases

- Promotional discount displays
- Promo code entry at checkout
- Marketing campaign coupons
- Loyalty reward vouchers
