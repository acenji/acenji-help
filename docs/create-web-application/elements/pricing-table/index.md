# Pricing Table

The **Pricing Table** element displays side-by-side pricing plan comparison cards. One plan can be highlighted as "featured" with a badge and accent border.

---

## Features

- **Side-by-side plan cards** in a responsive flex layout
- **Featured plan** highlighting with "Popular" badge
- **Feature checklist** with checkmark icons
- **CTA buttons** per plan
- **Accent color** theming

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Color for featured plan border and buttons | `#4285F4` |

---

## Plans

Each plan has:
- **Name** — plan name (e.g., "Starter", "Pro")
- **Price** — display price (e.g., "$29")
- **Period** — billing period (e.g., "/mo", "/yr")
- **Featured** — highlight as the recommended plan
- **Button Text** — CTA button label
- **Features** — comma-separated list of included features

Click **+ Add Plan** to add plans. Click **x** to remove.

---

## Use Cases

- SaaS pricing pages
- Subscription plan comparisons
- Service tier displays
- Membership level overviews
