# Mega Menu

The **Mega Menu** element provides a multi-column dropdown navigation menu. Hovering or clicking a top-level item reveals a panel with categorized links and icons.

---

## Features

- **Horizontal navigation bar** with top-level items
- **Multi-column dropdown** panels with headings and icon links
- **Full-width or dropdown** positioning
- **Simple link items** (no dropdown) for items without columns
- **Customizable colors** for background, text, and accent

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Hover and active color | `#4285F4` |
| **Background Color** | Dropdown panel background | `#ffffff` |
| **Text Color** | Menu text color | `#333333` |
| **Style** | Panel style: full-width or dropdown | `full-width` |

---

## Menu Items

Each menu item has:
- **Label** — top-level menu text

Items can have multi-column dropdowns configured with headings and links, or act as simple navigation links (no dropdown).

---

## Use Cases

- Main website navigation
- Product catalog navigation
- Documentation site menus
- Enterprise portal navigation
