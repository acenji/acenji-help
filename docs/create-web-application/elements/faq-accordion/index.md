# FAQ Accordion

The **FAQ Accordion** element displays an expandable list of questions and answers. Users click a question to reveal or hide its answer.

---

## Features

- **Expand/collapse** individual questions
- **Single or multi-expand** mode
- **Optional search bar** to filter questions
- **Optional numbering** for questions
- **Smooth toggle** with chevron icon

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Color for active question and icons | `#4285F4` |
| **Allow Multiple** | Allow multiple items open at once | `false` |
| **Show Search** | Display search bar to filter questions | `false` |
| **Show Numbers** | Number each question | `false` |

---

## Items

Each FAQ item has:
- **Question** — the question text
- **Answer** — the answer text (shown when expanded)

Click **+ Add** to add items. Click **x** to remove.

---

## Use Cases

- FAQ pages
- Product documentation
- Help center sections
- Knowledge base articles
