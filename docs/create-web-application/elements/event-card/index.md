# Event Card

The **Event Card** element displays event details in a styled card format with a date badge, time, location, and optional RSVP button.

---

## Features

- **Date badge** with month abbreviation and day number
- **Event title, time, and location** display
- **RSVP button** — interactive toggle for "Going" status
- **Category badge** — optional colored tag
- **Horizontal or vertical** layout
- **Description text** — optional truncated preview

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Title** | Event title text | `"Team Standup"` |
| **Date** | Event date (YYYY-MM-DD) | Today's date |
| **Time** | Event time string | `"10:00 AM"` |
| **Location** | Venue or address | `"Conference Room A"` |
| **Description** | Optional event description | — |
| **Accent Color** | Color for date badge and buttons | `#4285F4` |
| **Layout** | Card arrangement: horizontal or vertical | `horizontal` |
| **Category** | Optional category label (shown as badge) | — |
| **Show RSVP** | Display RSVP / Going button | `true` |
| **Show Location** | Display location with pin icon | `true` |
| **Show Description** | Display description text | `false` |

---

## Use Cases

- Event listing pages
- Conference schedules
- Meeting cards in dashboards
- Activity announcements

---

## Styling

The Event Card supports standard **Container** color and dimension settings:
- Background color, border, padding, margin
- Border radius and width
