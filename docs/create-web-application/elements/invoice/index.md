# Invoice

The **Invoice** element displays a professional invoice with company branding, line items, tax calculations, and payment actions.

---

## Features

- **Company header** with logo placeholder and invoice title
- **Invoice details** — number, date, due date
- **Bill To** section with client information
- **Line items table** — item name, quantity, unit price, total
- **Totals** — subtotal, tax, discount, grand total
- **Notes** section for payment terms
- **Action buttons** — Download PDF and Pay Now

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Accent Color** | Header and button color | `#4285F4` |
| **Show Logo** | Display company logo placeholder | `true` |
| **Show Pay Button** | Display Download PDF and Pay Now buttons | `true` |
| **Show Notes** | Display payment terms section | `true` |
| **Currency Symbol** | Currency prefix for amounts | `"$"` |

---

## Use Cases

- Invoice generation and display
- Billing dashboards
- Payment request pages
- Order confirmation receipts
