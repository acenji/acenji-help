# Testimonial

The **Testimonial** element displays a customer quote card with author details, star rating, and avatar.

---

## Features

- **Quote text** with decorative quotation marks
- **Author avatar** — image or colored initials circle
- **Star rating** (1-5 stars, optional)
- **Three layouts** — card, minimal, large-quote
- **Accent color** bar for visual emphasis

---

## Settings

| Setting | Description | Default |
|---------|-------------|---------|
| **Quote** | Testimonial text | Sample quote |
| **Author** | Author name | `"Jane Smith"` |
| **Role** | Author title/company | `"Product Manager, Acme Inc."` |
| **Rating** | Star rating 0-5 (0 = hidden) | `5` |
| **Accent Color** | Color for accent bar and avatar | `#4285F4` |
| **Layout** | Card style: card, minimal, or large-quote | `card` |
| **Show Avatar** | Display avatar circle | `true` |
| **Avatar URL** | Custom avatar image URL | — |

---

## Use Cases

- Customer reviews on landing pages
- Product testimonials
- Case study quotes
- Social proof sections
